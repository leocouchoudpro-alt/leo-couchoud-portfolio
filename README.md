# Léo Couchoud — Portfolio

Site portfolio personnel de Léo Couchoud, construit avec Next.js 15, TypeScript et Tailwind CSS.

## Stack

- **Framework** : Next.js 15 (App Router)
- **Langage** : TypeScript
- **Style** : Tailwind CSS v4
- **Déploiement** : Vercel

## Lancer en local

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).
